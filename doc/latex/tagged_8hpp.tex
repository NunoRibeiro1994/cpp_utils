\doxysubsection{Detailed Description}
Header file providing declaration \& definition of std\+::tagged$<$$>$ class template. 

\begin{DoxyAuthor}{Author}
João André
\end{DoxyAuthor}
std\+::tagged$<$$>$ is a very thin wrapper around an object, that provides a public id key for indexing/identification purposes. Similar to std\+::pair$<$$>$ or std\+::tuple$<$$>$, but with the significant caveat that it is implicitly constructibe from and convertible to the underlying object type, which means it can be used interchangeably in most situations.
