\doxysubsection{Detailed Description}
Base class for unique objects, holding a generic unique identifier (UUID). 

\begin{DoxyNote}{Note}
Useful to identify \& reference functional parametrizations of primitives, skills and memories.

Wraps around boost\+::uuids\+::uuid providing implicit conversion to std\+::string. 
\end{DoxyNote}
