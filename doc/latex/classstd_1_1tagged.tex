\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T, typename Key = string$>$\newline
class std\+::tagged$<$ T, Key $>$}
Associative type template, indented to add a key to type T, wherein a key \textquotesingle{}tag\textquotesingle{} is attached to an object. 

\begin{DoxyNote}{Note}
Similar to std\+::pair$<$$>$, but prioritizing T over the key\+: from the outside it behaves like T, is implicitly convertible to T (and therefore can be used interchangeably), but provides an additional key member, publicly accessible \& modifiable.

Design to (loosely) match STL guidelines/conventions. 
\end{DoxyNote}
