\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Command\+System = Spring\+Damper$>$\newline
class motion\+::\+Basic\+Policy\+Generator$<$ Command\+System $>$}
Generic base abstract class for open-\/ and closed-\/loop multidimensional motion policy generators. 

\begin{DoxyNote}{Note}
Assumes separate (shared) phase and command/motor dynamics as \mbox{\hyperlink{classmotion_1_1_dynamical_system}{Dynamical\+System}} derived implementations. Takes ownership of command/motor dynamics while using external phase supplier.

Also (optionally) allows feedback mechanisms represented w/ Policy\+Feedback class. 
\end{DoxyNote}
