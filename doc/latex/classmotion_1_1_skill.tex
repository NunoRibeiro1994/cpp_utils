\doxysubsection{Detailed Description}
Class describing a motion skill as a weigthed combination of multiple primitives, for variable output dimensions. 

\begin{DoxyNote}{Note}
Wraps around a weigth matrix which dynamically encodes the number of output dimensions. Can be used to extract multiple independent primitive outputs if an Identity matrix is used

Stores pointers to individual primitives {\itshape without taking ownership of them}. 
\end{DoxyNote}
