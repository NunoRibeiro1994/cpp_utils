\doxysubsection{Detailed Description}
Class implementing a versatile motion primitive as a linear combination of non-\/linear basis functions. 

\begin{DoxyNote}{Note}
Useful to encode trajectores of varying complexity in a parametrized policy representation. For higher order trajectories, higher number of basis can be used, at the cost of higher dimensionality. 
\end{DoxyNote}
