\doxysubsection{Detailed Description}
Class implementing a dynamic library of primitives. 

\begin{DoxyNote}{Note}
Wraps around a simple vector of \mbox{\hyperlink{classmotion_1_1_primitive}{Primitive}} instances, implementing mixing/averaging of different primitive for a specific dimension. 
\end{DoxyNote}
