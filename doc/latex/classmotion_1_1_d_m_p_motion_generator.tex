\doxysubsection{Detailed Description}
Generic multi-\/dimensional DMP-\/base motion generator, extending \mbox{\hyperlink{classmotion_1_1_basic_policy_generator}{Basic\+Policy\+Generator}}. Provides step-\/wise motion generation interface to an existing system/library of motion primitves. 

\begin{DoxyNote}{Note}
Implements {\itshape N} DMP transformation systems (wherein {\itshape N} matches desired output dimensions e.\+g. agent degrees-\/of-\/freedom) 
\end{DoxyNote}
