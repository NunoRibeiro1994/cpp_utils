\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Container, typename T = typename Container\+::value\+\_\+type$>$\newline
class std\+::cast\+\_\+iterator$<$ Container, T $>$}
Generic class describing a {\itshape random access} bidirectional iterator, with different container and dereference (target) type. Allows masking an interator to a different type when types are implicitly convertible. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Container} & Generic container of descriptor/tagged type instances. Must fit STL\textquotesingle{}s Container \& Sequence\+Container (partial) requirements. \\
\hline
{\em T} & Output/\textquotesingle{}public\textquotesingle{} type\\
\hline
\end{DoxyTemplParams}
\begin{DoxyNote}{Note}
T and Container\+::value\+\_\+type must be implicitly convertible.

Useful when handling types that are interchangeable (e.\+g. base/derived on polymorphic implementations).

Meets STL {\itshape Random\+Access\+Iterator} \href{https://cplusplus.com/reference/iterator/RandomAccessIterator/}{\texttt{ requirements}}, and is compatible with STL {\itshape Sequence\+Container} types. 
\end{DoxyNote}
