Detailed Description

template<typename CommandSystem = SpringDamper>
class motion::BasicPolicyGenerator< CommandSystem >

Generic base abstract class for open- and closed-loop multidimensional motion policy generators.

Note
Assumes separate (shared) phase and command/motor dynamics as DynamicalSystem derived implementations. Takes ownership of command/motor dynamics while using external phase supplier.
Also (optionally) allows feedback mechanisms represented w/ PolicyFeedback class.
