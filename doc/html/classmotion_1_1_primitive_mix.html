Detailed Description

Class describing a weigthed combination of multiple primitives, for variable output dimensions.

Note
Wraps around a weigth matrix which dynamically encodes the number of output dimensions. Can be used to extract multiple independent primitive outputs if an Identity matrix is used
Stores pointers to individual primitives without taking ownership of them.
