Detailed Description

Generic multi-dimensional DMP-base motion generator, extending BasicPolicyGenerator. Provides step-wise motion generation interface to an existing system/library of motion primitves.

Note
Implements N DMP transformation systems (wherein N matches desired output dimensions e.g. agent degrees-of-freedom)
