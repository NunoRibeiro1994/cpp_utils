Detailed Description

template<typename InputType>
class motion::BasicPolicyFeedback< InputType >

Generic base abstract class for implementing dynamical feedback mechanisms.

Note
Templated according to input data source for versatility and a more generic approach (e.g. ros::Sensor, etc.)
