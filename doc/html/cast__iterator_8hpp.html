Detailed Description

Header file providing declaration & definition of std::cast_iterator<> class template.

Author
João André

std::cast_iterator<> is a random access iterator class template derived from std::iterator<> that allows a different public value type than its underlying/source container, provided that they are implicitly convertible (effectively 'masks' underlying type). Intended to work with std::indexer<>, but was designed generically with a broader scope of application in mind, being compatible with STL sequence containers (std::vector, std::array, std::deque, std::list and potentially std::string)
